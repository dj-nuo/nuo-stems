# Download NUO-STEMS 4 beta

NUO-STEMS 4 is available for download on the following platforms:

- **Windows** - https://nuo-stems-4.fra1.cdn.digitaloceanspaces.com/NUO-STEMS-4-4.0.0-beta.54.exe.zip  
- **macOS** (Apple Silicon) - https://nuo-stems-4.fra1.cdn.digitaloceanspaces.com/NUO-STEMS%204-4.0.0-beta.54-arm64.dmg    
- **macOS** (Intel) - https://nuo-stems-4.fra1.cdn.digitaloceanspaces.com/NUO-STEMS%204-4.0.0-beta.54.dmg

Changelog is available [here](/docs/changelog).

## System Requirements

### GPU Acceleration

For optimal performance with GPU acceleration:

- **Windows**: NVIDIA GPU with 4GB or more VRAM
- **macOS**: Apple Silicon (M1/M2/M3 or later)

GPU acceleration is optional - NUO-STEMS 4 will work on CPU as well, though processing will be slower.

:::tip
All users who purchased NUO-STEMS 3 **after 1 November 2025 will automatically get a free upgrade** to NUO-STEMS 4. Just install NUO-STEMS 4 and your license will be activated automatically on the same computer where you have NUO-STEMS 3 installed.
:::
