NUO-STEMS 4 includes a demo mode that allows you to try the software before purchasing a license.

## Demo Mode Limitations

Only **20 seconds of audio** is processed starting from the middle of the track. There is no limit on the number of tracks you can process in demo mode.

:::warning
**Traktor [natively linked STEMS](/docs/getting-started/native-traktor-pro-4) are not allowed in demo mode** because linking short 20-sec audio tracks to full length originals doesn't make sense.

However, you can still use [Legacy / Export Mode](/docs/getting-started/legacy-traktor) to create .stem.m4a files or [DAW / Non-Traktor mode](/docs/getting-started/daw-non-traktor) to create .wav files for testing purposes.
:::

## Activating Full Version

To unlock all features, including Traktor STEMS creation:

1. **Purchase a license** - NUO-STEMS 4 requires a separate license (NUO-STEMS 3 licenses do not work with version 4 - see [FAQ](/docs/faq))
2. **Patreon subscribers** - If you're a Patreon subscriber, your license will be activated automatically when you install NUO-STEMS 4.
