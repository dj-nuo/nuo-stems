This utility extracts individual stems from **Traktor Pro 4** `.stem.m4a` / `.stem.mp4` containers (TP4 legacy or natively linked stem files).

Useful when you want stems from those files for DJ software other than Traktor (for example Rekordbox) or for DAWs (Ableton Live, Logic Pro, etc.).

:::tip
For **`.serato-stems`** files from NUO-STEMS 4 Serato integration, use [Extract individual stems from .serato-stems](/docs/utilities/extract-individual-stems-from-serato-stems) instead (stems are saved as **`.mp3`** files).
:::

:::tip
Extracting stems from `.stem.m4a` / `.stem.mp4` is quicker than re-generating them using NUO-STEMS 4.

But it also means that quality of the stems will be lower than if you re-generate them using NUO-STEMS 4 in [DAW mode](/docs/getting-started/daw) and save results as `.wav` files, because `.stem.m4a` / `.stem.mp4` are usually encoded as AAC (defined in [settings](/docs/settings/traktor-pro)). You can change this in NUO-STEMS 4 Settings to use ALAC encoding, but it has its own drawbacks.
:::

## Overview

When you extract WAV files from TP4 `.stem.m4a` / `.stem.mp4`:
- Individual stem files are extracted as separate WAV files
- Each stem (vocals, drums, bass, other, master) is saved as its own WAV file
- Files are saved to your specified output folder
- Original `.stem.m4a` / `.stem.mp4` files remain unchanged

## Multi-track playback
Just as a handy feature, you can play back the stems as multi-track audio. With ability to solo/mute individual stems.
