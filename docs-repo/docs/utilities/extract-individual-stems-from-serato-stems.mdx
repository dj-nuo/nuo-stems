This utility extracts individual stems from **`.serato-stems`** files produced by NUO-STEMS 4 when using [Serato mode](/docs/getting-started/serato).

Use it when you want stem **`.mp3`** files for a **DAW**, **another DJ app**, or any workflow outside Serato DJ — without re-running full separation on the original track.

It is also useful when you want to inspect the file metadata and confirm whether a `.serato-stems` file was created by NUO-STEMS 4 or by Serato itself.

:::tip
Extracting stems from an existing `.serato-stems` file is faster than generating stems again from the original audio in NUO-STEMS 4.

Output is **MP3** (lossy). That is similar in spirit to [Extract individual stems from TP4 `.stem.m4a`](/docs/utilities/extract-individual-stems-from-stem-m4a), where TP4 container stems are decoded to files for external use. If you need **uncompressed WAV** stems, use [**DAW mode**](/docs/getting-started/daw) and export from the original file instead.
:::

## Overview

When you extract stems from `.serato-stems`:

- Each stem is written as its own **`.mp3`** file (vocals, drums, bass, other).
- Files are saved to the output folder you choose in the utility.
- Original `.serato-stems` files are not modified.

## How to use

1. Open NUO-STEMS 4 and go to the **Utilities** section.
2. Choose **Extract individual stems from .serato-stems**.
3. Select the `.serato-stems` files or folder you want to process.
4. Choose the output folder and run extraction.

## Multi-track playback
Just as a handy feature, you can play back the stems as multi-track audio. With ability to solo/mute individual stems.
