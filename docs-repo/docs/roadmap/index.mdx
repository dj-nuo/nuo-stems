    - In "native TP4" mode, when you add tracks from "Traktor Pro 4" tab as "traktor" tracks into queue - only "native linked" stems will be generated. No .stem.m4a in the output folder.

    - In "legacy" mode, when you add tracks from "Traktor Pro 4" tab into queue - they are added as regular filesystem tracks. .stem.m4a will be generated.
    - In "legacy" mode, when you add tracks just by drag-and-dropping from the file system - they are added as regular filesystem tracks. .stem.m4a will be generated.
    - .wav or instrumental+acapella.mp3 will stay "as-is" and won't be affected by this setting
